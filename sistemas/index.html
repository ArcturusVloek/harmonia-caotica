Sistemas

O arquivo que reunirá as regras oficiais para criação de personagens, progressão, milagres, combate e demais estruturas do RPG.

Esta seção foi criada para receber os sistemas de Harmonia Caótica quando cada conjunto de regras estiver revisado, testado e pronto para publicação.

Conteúdo em preparação. Nenhuma versão provisória será apresentada como regra oficial.

Uma estrutura em construção

Os sistemas serão publicados gradualmente e organizados para consulta rápida. Cada página deverá explicar somente aquilo que o jogador precisa utilizar, mantendo regras administrativas e situações excepcionais em seções próprias.

Até a conclusão dessa revisão, esta página permanecerá como a entrada oficial da futura área de sistemas.
