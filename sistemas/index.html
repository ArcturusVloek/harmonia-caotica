Você não precisa estudar toda a biblioteca para começar. Escolha o caminho que corresponde ao que procura.

Harmonia Caótica possui regras profundas, mas a complexidade deve aparecer apenas quando ela for necessária. Novos jogadores recebem orientação; leitores experientes mantêm acesso direto a tudo.

Transição entre versões

Para quem está retornando

A identidade de Vaelora permanece. A antiga estrutura mecânica não deve ser misturada às regras atuais.

História, personalidade, vínculos, Divindade, Domínio e conceito podem ser preservados quando continuam canônicos. Vida, valores, alcance, duração e funcionamento dos poderes precisam ser reconstruídos.

Abrir guia de retorno

Publicação responsável

O restante continua em desenvolvimento

Somente conteúdo aprovado é oficial

Novos capítulos são discutidos, auditados e aprovados antes de aparecerem na Biblioteca.

Os próximos temas serão apresentados discretamente conforme se tornarem relevantes. A página principal não exibirá uma longa lista de regras ainda inexistentes.

Princípio editorial

Profundidade sem barreiras

O sistema continuará oferecendo construções complexas para jogadores experientes, mas não exigirá que novos jogadores compreendam todas elas antes de participar.

Orientação para quem está começando.

Acesso direto para quem sabe o que procura.

Profundidade completa para quem deseja dominar o sistema.
